# Early Use Cases

This folder collects early, use-case-oriented requirements for the template example.

## Goal

The files in this folder describe the need, expectation, and value of the requirements phase, without pre-empting business architecture or technical implementation.

## Structure

- `order.md`: placing an order and receiving order status
- `preparation.md`: preparing coffee after order receipt
- `delivery.md`: delivering the finished coffee and reporting completion

## Origin

The use cases were migrated from the previous versioned individual requirements and organised into the new requirements structure.
