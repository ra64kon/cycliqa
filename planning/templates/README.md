# Sprint Templates

This directory contains sprint templates derived from `planning/WORKFLOW.md`.

## Purpose

- Provide reusable structure for new sprint files
- Document the common thread from pre-planning through planning to completion
- Ensure consistent QA and release sections for future sprints

## Files

- `sprint_pre_planning.template.md`: Template for prioritised, sprint-ready backlog candidates in `1_plan`
- `sprint_planning.template.md`: Template for the concretely elaborated sprint in `2_work`
- `sprint_done.template.md`: Template for the completed sprint in `3_done`
