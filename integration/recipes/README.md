Demo interface for external integration

Source:

https://dummyjson.com/recipes

This folder contains a small reference integration for the DummyJSON Recipes API.
The OpenAPI specification created for it is located under `api/recipes.openapi.yaml`.

Covered in particular:

- Reading all recipes including pagination and sorting
- Recipe search
- Fetching individual recipes
- Filtering by tag and meal type
- Simulated write operations for add, update, and delete
